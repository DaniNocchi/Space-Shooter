		  Process to make a new powerup
1- Get the Idea
3- make an sprite to it
4- add the sprite in the sprite list in powerup.gd
5- add its effect in powerup.gd
6- add it in the list in powerup_spawner.gd
7- make it actually work


		   in case of including a timer
8- go to progressbarpowerup.gd and add the timerTime
9- in the same script, add the timer timeout effect 
   (disable the powerup after ending the timer)


	in case of using a new controller variable
10- Create the variable in the game_controller.gd
11- reset it in the r_game.gd
